How to install ?
----------------
1 - Locate the project folder of your current UE4 project.
2 - Extract the "MaterialLayering" folder into the "Content" folder of your project.
3 - Launch the editor (version 4.12 minimum) with your project.


How to use ?
----------------
1 - In the editor, go to "MateriaLayering" then "Shader" and find "mat_layering_master"
2 - Right click on "mat_layering_master" and choose "Create Material Instance"
3 - Open the new Material Instance and start tweaking its parameters.

see our documentation for more informaiton : 
https://support.allegorithmic.com/documentation/display/SPDOC/Dynamic+Material+Layering
